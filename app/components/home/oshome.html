Open-source Snake AI Platform

Welcome to OpenSnap the programming platform where you finally have the opportunity to write your own implementation of a snake's brain.

To get started click the create icon on the sidebar and you can start experiencing with writing your AI code and test it out in a live snake environment. You can save your snippet any time and update it later.
To load the existing snippets click the codes button on the sidebar, choose a snippet you like and click the arrow icon to load it.

Happy Snapping!
